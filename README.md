FLock : Floating Lock
======
Refer to README.en.md to view the English version of README

簡介
------
浮動鎖屏按鈕。

提供一個浮動的鎖屏按鈕。

軟件鎖屏按鈕可以讓你不必再擔心實體按鈕的壽命問題。
浮動按鈕可以讓你在需要鎖屏時保持在當前工作程序中，而不必返回到初始屏幕。
